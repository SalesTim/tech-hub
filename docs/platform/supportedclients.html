# Microsoft Teams clients

SalesTim is a web application and a suite of integrated services accessible from Microsoft Teams. SalesTim can be accessed from the Microsoft Teams desktop client and from the Microsoft Teams web client. Get clients for Microsoft Teams (opens new window)

Microsoft Teams mobile client

As of today, the Microsoft Teams mobile client (Android and iOS) is not yet supported.
