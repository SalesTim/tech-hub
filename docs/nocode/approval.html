# Team creation approval workflow

This article describes how to configure SalesTim to create an approval workflow for your teams creation requests with your own internal email as a sender to approvers.

# Abstract

To implement its approval workflow, SalesTim relies on Outlook Actionable Messages (opens new window) to help approvers manage their approval requests right from their inbox. Outlook Actionable messages are emails that embed Adaptive Cards (opens new window) in a very secure way, by enforcing both end-user authentication and phishing prevention.

# More about Actionable Messages

# what about security?

  • End-user authentication is entirely managed by the Outlook client (Desktop, Web and Mobile) by providing to the actionable message the required token.
  • Phishing prevention is ensured by using a Card Signing (opens new window) mechanism

# How does it works?

Actionable messages security is guaranteed by:

  • The fact that the actionable message could only be used from the context of a secured Outlook client.
  • A sender verification is enforced using signed cards. See: Sender verification (opens new window)
  • Requests sent by the actionable message are verified to ensure that they originate from Microsoft. See: Verifying that requests come from Microsoft (opens new window)
  • The token provided by Outlook to the actionable message is used to verify the end-user identity. See: Verifying the identity of the user (opens new window).

Learn more by reading Security requirements for actionable messages in Office 365 (opens new window).
