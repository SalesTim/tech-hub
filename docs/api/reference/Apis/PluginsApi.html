# PluginsApi

All URIs are relative to https://api.salestim.io/v1.0

Method HTTP request Description
createPlugin POST /plugins Create a new plugin
deletePlugin DELETE /plugins/{id} Delete a plugin
getPlugIns GET /plugins Get all plugins in a tenant
regeneratePluginSecret POST /plugins/{id}/regenerateSecret Regenerate a plugin secret
updatePlugin PUT /plugins/{id} Update a plugin

# createPlugin

Plugin createPlugin(inlineObject1)

Create a new plugin

Create a new plugin and get its generated ID and secret.

# Parameters

Name Type Description Notes
inlineObject1 InlineObject1

# Return type

Plugin

# Authorization

bearerAuth

# HTTP request headers

  • Content-Type: application/json
  • Accept: application/json
