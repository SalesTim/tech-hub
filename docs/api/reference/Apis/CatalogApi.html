# CatalogApi

All URIs are relative to https://api.salestim.io/v1.0

Method HTTP request Description
getCatalogTemplates GET /catalog/templates Get all templates from your corporate catalog
installTemplateFromStore POST /catalog/templates/installFromStore Install a template from the public template store to your corporate catalog

# getCatalogTemplates

List getCatalogTemplates()

List

# Authorization

  • Content-Type: Not defined
