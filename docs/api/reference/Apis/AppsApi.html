# AppsApi

All URIs are relative to https://api.salestim.io/v1.0

Method HTTP request Description
createApp POST /apps Create a virtual app
deleteApp DELETE /apps/{id} Delete a virtual app
getApps GET /apps Get all virtual apps in a tenant
regenerateAppSecret POST /apps/{id}/regenerateSecret Regenerate a virtual app secret
updateApp PUT /apps/{id} Update a virtual app

# createApp

App createApp(app)

Create a virtual app

Create a new virtual app and get its generated ID and secret.

# Parameters

Name Type Description Notes
app App List of properties to be updated.

# Return type

App

# Authorization

bearerAuth

# HTTP request headers

  • Content-Type: application/json
  • Accept: application/json

# deleteApp

deleteApp(id)

Delete a virtual app

Delete a virtual app

# Parameters

Name Type Description Notes
id String ID of the virtual app to delete [default to null]

# Return type

null (empty response body)

# Authorization

bearerAuth

# HTTP request headers

  • Content-Type: Not defined
  • Accept: Not defined
