# Properties

Name Type Description Notes
name String The virtual app name [optional] [default to null]
description String The virtual app description [optional] [default to null]
createdBy UUID The ID (Fron Azure AD) of the user that created the virtual app [optional] [default to null]
createdAt Date The date of creation of the virtual app [optional] [default to null]
id UUID The virtual app ID, auto-generated at creation [optional] [default to null]
secret String The virtual app secret, auto-generated at creation. It may also be regenerated later. Be careful, this property is only available at creation or after an explicit regeneration. [optional] [default to null]
enabled Boolean The virtual app status, enabled or disabled [optional] [default to true]
