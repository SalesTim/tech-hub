# Documentation for API Endpoints

All URIs are relative to https://api.salestim.io/v1.0

Class Method HTTP request Description
CatalogApi getCatalogTemplates GET /catalog/templates Get all the team templates from your catalog (Restricted through RBAC to the following roles: "Global admin", "Teams service admin", "Catalog Manager")
CatalogApi getMyCatalogTemplates GET /me/catalog/templates Get my teams templates
JobsApi getJob GET /jobs/{JobId} Get information about a job
LabelsApi getMyLabels GET /me/labels Get my sensitivity labels
TeamsApi createTeamProvisioningJob POST /powerplatform/teams/provisioning Create a new team based on a template
